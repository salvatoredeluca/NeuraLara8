# neura_robot_description

This project contains all necessary information about lara(3,5,8,10,15), maira(6S,6M,6L,6JR,7S,7M,7L,7JR) and liwa products.
Each robot has 3 folders collision, meshes and urdf.
The urdf folder has kinematic and dynamics xacro files, the robot urdf, limits.xacro and macro.xacro.
